# Student Information System

## Objective

Build a command-line application that collects student information from the user and displays it in a well-formatted report.

This project reinforces concepts learned in:

- Chapter 2 – Variables & Memory
- Chapter 3 – Data Types
- Chapter 4 – Operators
- Chapter 5 – Input & Output

---

## Problem Statement

Create a command-line application that accepts student information from the user, performs basic calculations, and displays a formatted report.

Unlike previous projects, this application should not use hardcoded values. All information must be entered by the user during program execution.

---

## Concepts Covered

- Variables
- Data Types
- User Input
- Output Formatting
- Arithmetic Operators
- Type Conversion
- f-Strings

---

## Functional Requirements

The application should accept the following details from the user.

### Personal Information

- Student ID
- Full Name
- Age
- Gender
- Date of Birth

---

### Academic Information

- Course
- Branch
- Semester
- College Name

---

### Contact Information

- Email
- Mobile Number
- City
- State
- Country

---

### Marks

Accept marks for five subjects.

- Subject 1
- Subject 2
- Subject 3
- Subject 4
- Subject 5

---

## Calculations

Calculate and display:

- Total Marks
- Average Marks
- Percentage

Assume each subject is out of 100 marks.

---

## Program Requirements

The program should:

- Accept all values using `input()`
- Convert values to appropriate datatypes
- Use meaningful variable names
- Display output using formatted strings
- Organize the report into sections
- Display calculated values
