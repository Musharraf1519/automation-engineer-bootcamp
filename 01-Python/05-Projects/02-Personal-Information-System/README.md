# Personal Information System

## Objective

Build a command-line application that demonstrates the practical use of Python data types by storing, displaying, and processing personal information.

---

## Problem Statement

Create a command-line application that stores a person's information using appropriate Python data types.

The application should display all stored information along with calculated values and the datatype of every variable.

---

## Concepts Covered

- Integer
- Float
- String
- Boolean
- NoneType
- Complex
- type()
- Type Conversion
- Formatted Output (f-Strings)

---

## Requirements

Store the following information:

- Full Name
- Age
- Height (in cm)
- Weight (in kg)
- Gender
- City
- Country
- Email
- Phone Number
- Occupation
- Monthly Income
- Is Married
- Blood Group
- Passport Number
- Emergency Contact
- Manager (Initially None)

---

## Calculations

Calculate:

- Annual Income
- Age After 10 Years
- Height in Meters
- BMI

Formula

BMI = Weight / (Height in meters × Height in meters)

---

## Program Requirements

The program should:

- Use meaningful variable names.
- Print all information using f-strings.
- Display the datatype of every variable.
- Display calculated values.
- Organize output neatly.
