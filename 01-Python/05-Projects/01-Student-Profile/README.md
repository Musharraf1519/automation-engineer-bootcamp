# Student Profile

## Objective

Learn how variables are used in a real-world Python program.

---

## Problem Statement

Create a command-line application that stores and displays a student's profile.

The application should:

- Store student details.
- Display the information neatly.
- Calculate derived values.
- Demonstrate proper variable naming.

---

## Concepts Covered

- Variables
- Data Types
- Arithmetic Operators
- Constants
- Formatted Output (f-Strings)

---

## Requirements

The application should store:

- Student ID
- Name
- Age
- Gender
- Course
- College
- City
- Email
- Phone Number
- Semester
- CGPA
- Annual Fee

It should calculate:

- Monthly Fee
- Age Next Year
- Percentage from CGPA
