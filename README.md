# Automation Engineering Bootcamp

A comprehensive, project-based learning repository designed to help developers, students, and automation enthusiasts become proficient Automation Engineers through structured learning, practical exercises, and real-world projects.

---

## About

This repository follows a structured roadmap covering software engineering fundamentals, automation tools, backend concepts, infrastructure, and AI-powered automation.

Every topic is designed to be learned through:

- Detailed Notes
- Reference Code
- Practice Exercises
- Complete Solutions
- Hands-on Projects

The objective is not only to learn concepts but also to build a production-quality portfolio throughout the journey.

---

## Learning Objectives

By completing this bootcamp, you will be able to:

- Write clean and maintainable Python code
- Build automation scripts
- Work with Git and GitHub
- Understand Linux fundamentals
- Consume and develop REST APIs
- Work with SQL and PostgreSQL
- Build Dockerized applications
- Design automation workflows using n8n
- Integrate AI into automation systems
- Build AI Agents and RAG applications
- Deploy production-ready automation projects
